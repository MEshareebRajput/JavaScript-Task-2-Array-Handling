Understanding Array Element Swapping

In JavaScript, an array is a collection of values stored under a single variable name. Swapping elements of an array is a common task in array handling, which can be done using different methods.

In this task, we are swapping the first and last elements of an array using a clever one-liner, demonstrating the power of JavaScript's syntax.

JavaScript Implementation

Below is the JavaScript code that swaps the first and last elements of the array: 


let arr = ["Mango", "Orange", "Grapes"];

arr[0] = [arr[2], arr[2] = arr[0]][0];

console.log(arr); // Outputs: ["Grapes", "Orange", "Mango"]

Results

Results:

Original Array: Mango, Orange, Grapes

Swapped Array: Grapes, Orange, Mango
